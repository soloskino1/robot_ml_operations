# robot_ml_operations

Video Presentation - https://www.youtube.com/watch?v=2lcdK3jchYM

Task 1: ANN Regression for robot arm control
inverse_kinematics.ipynb - Performed experiments on learning inverse kinematics transformations of a robot arm using the Multilayer Perception (MLP) Regressor from sciki-learn. 

Task 2: Classification and PCA on mobile robot sensory data
robot_sensing.ipynb - Performed experiments on classification of different floor types on the CareerCon dataset using the MLP classifier from scikit-learn

Task 3: Q-learning for mobile robot navigation
maze_navigation.ipynb - Performed experimetns on maze navigation using Q-learning
